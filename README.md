# darkly

> Reverse engineering d'une application web dans un conteneur — Projet 42

## Description

Projet de sécurité / reverse engineering : analyse d'une application web fournie dans une machine virtuelle Docker, découverte de vulnérabilités et exploitation contrôlée.

Deux variantes du binaire cible : `avec_flag` et `sans_flag`.

## Stack

`Docker` · `Web` · `Reverse engineering` · `Sécurité`

## Compétences

`Analyse binaire` · `Web security` · `Docker`
